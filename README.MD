# рҹҡҖ AI Testing Project (Playwright + IA)

Framework de pruebas automatizadas con Playwright + Inteligencia Artificial, capaz de detectar cambios en APIs, aprender del historial, generar escenarios automГЎticamente y ejecutar validaciones inteligentes.

---

## рҹ§  ВҝQuГ© hace este proyecto?

Este proyecto transforma el testing tradicional en un sistema inteligente que:

- Detecta endpoints automГЎticamente  
- Genera escenarios de prueba con IA  
- Ejecuta pruebas multi-endpoint  
- Detecta cambios en APIs  
- Aprende del historial (auto-adaptativo)  
- Genera asserts dinГЎmicos  
- Clasifica riesgo automГЎticamente  
- Genera dashboard visual  
- Manejo de ambientes  

---

## рҹҢҚ Manejo de ambientes

El sistema soporta mГәltiples entornos:

- mock вҶ’ Usa WireMock (local)  
- dev вҶ’ API de desarrollo  
- qa вҶ’ API de pruebas  
- prod вҶ’ API productiva  

---

## вҡҷпёҸ ConfiguraciГіn

```js
const ENV = process.env.ENV || 'mock';

const CONFIG = {
  mock: 'http://localhost:3000',
  dev: 'https://dev.api.bank.com',
  qa: 'https://qa.api.bank.com',
  prod: 'https://api.bank.com'
};

export const BASE_URL = CONFIG[ENV];
```

---

## рҹ§Ә EjecuciГіn por ambiente

### рҹ”ө Mock (con WireMock)

```bash
set ENV=mock
npx playwright test
```

- Levanta WireMock automГЎticamente  
- Usa datos simulados  

---

### рҹҹў Dev / QA / Prod (sin WireMock)

```bash
set ENV=dev
npx playwright test
```

- NO levanta WireMock  
- Consume APIs reales  

---

## рҹҗі Control automГЎtico de WireMock

El sistema decide automГЎticamente:

- Si ENV = mock вҶ’ levanta WireMock  
- Si ENV вү  mock вҶ’ no levanta nada  

---

## рҹ”Ҙ Flujo del sistema

1. Detecta endpoints automГЎticamente  
2. Genera escenarios con IA  
3. Ejecuta pruebas dinГЎmicas  
4. Compara con baseline  
5. Detecta cambios  
6. Aprende del historial  
7. Genera nuevos tests  
8. Ejecuta asserts inteligentes  
9. Clasifica riesgo  
10. Genera dashboard
